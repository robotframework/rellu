===========
Rellu 1.0.1
===========


.. default-role:: code


Rellu provides utilities to ease creating releases.
Rellu 1.0.1 is a new hotfix release with fix to setting version
incorrectly if VERSION_PATTERN did not match.

All issues targeted for Rellu v1.0.1 can be found from the
`issue tracker`_.

If you have pip_ installed, just run

::

   pip install --upgrade rellu

to install the latest available release or use

::

   pip install rellu==1.0.1

to install exactly this version. Alternatively you can download the source
distribution from PyPI_ and install it manually.

Rellu 1.0.1 was released on Friday December 5, 2025.
