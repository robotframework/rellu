# Rellu 0.8.0


Rellu provides utilities to ease creating releases.

All issues targeted for Rellu v0.8.0 can be found from the
[issue tracker](https://github.com/robotframework/rellu/issues?q=milestone%3Av0.8.0).

**REMOVE `--pre` from the next command with final releases.**
If you have `pip` installed, run:

```bash
pip install --pre --upgrade rellu
```

to install the latest available release or use

```bash
pip install rellu==0.8.0
```

to install exactly this version. Alternatively you can download the source
distribution from [PyPI](https://pypi.python.org/pypi/rellu) and install it manually.

Rellu 0.8.0 was released on Sunday November 23, 2025.
