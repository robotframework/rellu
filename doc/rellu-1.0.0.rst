===========
Rellu 1.0.0
===========


.. default-role:: code


Rellu provides utilities to ease creating releases.
Rellu 1.0.0 is a new release with which is the last
release to support restructured text format.

All issues targeted for Rellu v1.0.0 can be found from the
`issue tracker`_.

If you have pip_ installed, just run

::

   pip install --upgrade rellu

to install the latest available release or use

::

   pip install rellu==1.0.0

to install exactly this version. Alternatively you can download the source
distribution from PyPI_ and install it manually.

Rellu 1.0.0 was released on Sunday November 30, 2025.
